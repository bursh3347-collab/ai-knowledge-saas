# 🏗️ 叉盗行的开源项目工作空间

## 分支说明

| 分支 | 内容 | 状态 |
|------|------|------|
| `awesome-index` | 🗂️ GitHub开源项目全领域分类索引（26个分类） | ✅ 可用 |
| `tiangong-engine-v3` | 🔍 天工引擎 v3.0 — AI代码猎手脚本 | ✅ 可用 |
| `main` | 项目总索引（本页） | ✅ |

## 快速开始

### 查看分类索引
```bash
git checkout awesome-index
cd awesome-index
# 26个分类目录，每个目录包含Awesome列表链接+顶级项目表格
```

### 运行天工引擎
```bash
git checkout tiangong-engine-v3
cd tiangong-engine
npm install
npm run hunt:all
npm run batch -- --top=20
npm run report
```
